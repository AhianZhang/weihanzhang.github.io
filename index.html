个人博客是什么

个人博客，是互联网发展的必然产物。博主可以每天在这个平台上发表公开文章、浏览文章、评论和删除文章。

对于公司来说， 博客能够体现他们团队的活力和价值， 也能够帮助浏览者获取自己想要的信息； 但是对于个人来说， 一个优秀的博客能够提升自己的知名度， 帮助自己积累技术或者生活上的所见所得， 个人博客可以依托于如掘金、云栖社区、Medium 等国内外知名的社区； 也可以是 GitHub Page、WordPress、Solo 等通过简单部署就能搭建出来的个人博客。

为什么要做个人博客

一方面，为了巩固所学的知识，查漏补缺。 另一方面，在社区写博客固然方便简单，但是会有很多限制，比如有的内容不适合在社区上发表或者想要自定义一些模块， 所以自己搭建了一个 Java 语言的博客系统是一个不错的选择。

怎样做个人博客

参考各种博客的架构和前端样式， 决定使用以下配置:

  • 编辑器： IDEA
  • 前端： BootStrap / JQuery / EasyUI
  • 后端： Spring / SpringMVC / MyBatis
  • 数据库：MySQL

先将后台模块建立起来，再建立前台页面，最后前后端联调。

引言

这是一个互联网的时代, 博客系统只是其中的一个产物， 像人工智能、大数据、区块链技术这些大的门类都成为了国家重点关注的对象， 但是在当下浮躁的社会，我们最需要的就是沉着冷静， 此时博客就发挥了很大的作用， 它可以帮助我们沉淀自己，在写博客的时候 我们可以组织语言并且可以发现新的问题或者解决思路。 做一个个人博客的 想法很早就有了，网上的博客就那么几种， 用 Java 实现的就更少了，所以我就使用 Java 自己实现了一个博客。

分析阶段

技术可行性

Java是一种强类型的高级语言独立运行与JVM虚拟中实现跨平台， 其次Java的扩展性很强、社区活跃，是市场需求以及公司使用的第一大语言， 所以开发出来的Java版个人博客既有实用价值又有研究价值。

社会可行性

由于目前社会的生活节奏较快、方式单一、压力较大，有效的学习时间少之又少，此时写博客可以帮助人们活跃思维， 抒发情感或者记录所学所见的知识，有利于形成良好的社会风气。

经济可行性

因为本系统属于个体开发并且非商业用途，遵循 MIT 协议，几乎不需要成本，但是却可以巩固知识和帮助他人。 所以这是一个开发风险极低、收益极大的系统。

需求分析

为了实现响应式的页面, 该博客系统前台页面使用 BootStrap 做布局； 后台最重要的是数据处理， 所以使用封装 JQuery 的 EasyUI 做布局； 富文本编辑器采用 editormd ，贴合实际环境； 为了加快用户的读取速度， 在首次访问后会将公共数据放到 application 中； 使用 Shiro 做为博主登录的安全框架； 用户评论系统采用第三方的搜狐畅言插件。
